## 🚀 部署建议

### 前端
```bash
cd frontend
npm install
npm run build
```

### 后端
```bash
cd backend
mvn clean package
java -jar target/subtitle-generator-backend-1.0.0.jar
```

### 环境变量
```bash
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=subtitle_db
DB_USER=subtitle
DB_PASSWORD=subtitle123

# JWT 配置
JWT_SECRET=YourSecretKeyForJWTTokenGenerationMustBeVeryLong123456
JWT_EXPIRATION=86400000

# 应用配置
SERVER_PORT=8081
```
